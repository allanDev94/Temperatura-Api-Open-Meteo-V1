
Este proyecto es una aplicación web que permite consultar la temperatura actual de cualquier ciudad.

El usuario ingresa el nombre de una ciudad en un buscador, y la aplicación se encarga de obtener automáticamente sus coordenadas (latitud y longitud). Luego, con estos datos, se consulta una API externa (Open-Meteo) para mostrar el clima actual.

El objetivo principal del proyecto es aprender a trabajar con APIs, manejar datos en JavaScript y aplicar buenas prácticas básicas de desarrollo web.


## ▶️ ¿Cómo navegar y ejecutar el código?

Para usar la aplicación, sigue estos pasos:

1. Descargar o clonar el proyecto en tu computador
2. Abrir la carpeta del proyecto
3. Hacer doble clic en el archivo `index.html` o abrirlo con un navegador
4. Ingresar el nombre de una ciudad en el buscador
5. Presionar el botón "Consultar"
6. Visualizar la temperatura y datos en pantalla

### 🧭 Navegación dentro de la app

- Campo de texto: permite ingresar la ciudad
- Botón "Consultar": realiza la búsqueda del clima
- Sección de resultados: muestra la temperatura y cambia de color según el clima

No se requiere instalación de programas adicionales ni servidor, ya que la aplicación funciona completamente en el navegador.
